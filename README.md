# Deutsche Beschreibung

## Was ist der Bingo Generator?

Der **Bingo Generator** ist eine Desktop-Anwendung zum Erstellen individueller Bingo-Spielbögen.  
Er eignet sich besonders für Hochzeiten, Geburtstage, Feiern, Teamevents, Kennenlernspiele und andere Veranstaltungen, bei denen Gäste auf spielerische Weise miteinander ins Gespräch kommen sollen.

Die App erstellt aus frei definierbaren Fragen und Kategorien automatisch druckbare Bingo-Karten. Jede Karte kann unterschiedlich gemischt werden, während wichtige Vorgaben wie Kategorien, Mindestanzahl bestimmter Fragetypen oder das Layout erhalten bleiben.

## Typischer Ablauf

### 1. Kategorien anlegen

Fragen werden in Kategorien organisiert, zum Beispiel:

- Brautpaar
- Schule und Studium
- Reisen
- Familie
- Alltag
- Freizeit

Für jede Kategorie kann festgelegt werden, ob sie aktiv ist und wie viele Fragen daraus auf jedem Spielbogen erscheinen sollen. So lässt sich steuern, dass bestimmte Themen immer vorkommen, andere aber nur gelegentlich oder zum Auffüllen verwendet werden.

### 2. Fragen bearbeiten

Innerhalb jeder Kategorie können beliebig viele Fragen eingetragen werden.  
Die Fragen sollten so formuliert sein, dass sie auf dem Spielbogen den Satz ergänzen:

> Finde jemanden, der oder die ...

Beispiel:

> mit dem Brautpaar verwandt ist

Auf dem Spielbogen entsteht daraus sinngemäß:

> Finde jemanden, der oder die mit dem Brautpaar verwandt ist.

### 3. Spiel konfigurieren

Im Spielbereich werden die grundlegenden Einstellungen festgelegt:

- Größe des Bingo-Rasters, zum Beispiel 5 × 5
- Anzahl der zu erzeugenden Spielbögen
- optionaler Zufalls-Seed für reproduzierbare Ergebnisse
- Ausgabeformat und Speicherort

Der optionale Zufalls-Seed sorgt dafür, dass derselbe Kartensatz später erneut erzeugt werden kann. Bleibt das Feld leer, werden die Karten jedes Mal neu gemischt.

### 4. Layout gestalten

Im Layoutbereich kann das Design der Spielbögen angepasst werden:

- Titel
- Untertitel
- Datum
- Intro-Text direkt über dem Spielfeld
- Schriftarten, Schriftgrößen und Farben
- Seitenränder
- Feldgrößen und Rahmen
- Eckbilder für ein persönliches Design

Die mitgelieferten floralen Eukalyptus-Grafiken können direkt verwendet oder durch eigene Bilder ersetzt werden.

### 5. Vorschau prüfen

Die Vorschau zeigt eine visuelle Darstellung des aktuellen Layouts auf einem weißen Blatt.  
Sie hilft dabei, Abstände, Überschriften, Raster, Bilder und Textgrößen vor dem Export zu kontrollieren.

Die Vorschau kann gezoomt und gescrollt werden. Sie dient als Layout-Näherung; je nach Ausgabedatei und Drucker können kleine Unterschiede entstehen.

### 6. Exportieren

Die fertigen Bingo-Spielbögen können als Datei exportiert werden:

- **Word-Datei (.docx)** zur weiteren Bearbeitung
- **PDF-Datei (.pdf)** zum direkten Drucken oder Versenden

Vor dem Drucken sollte die Datei einmal kontrolliert werden, besonders wenn eigene Bilder, sehr große Schriftgrößen oder sehr kleine Seitenränder verwendet werden.

## Sprachen

Die Sprache der Benutzeroberfläche kann über Sprachdateien geändert werden.  
Weitere Sprachen können ergänzt werden, indem eine vorhandene Sprachdatei kopiert, übersetzt und unter neuem Namen gespeichert wird.

## Speicherung

Einstellungen und Kategorien werden lokal gespeichert.  
Änderungen an Kategorien bleiben während der Bearbeitung erhalten und werden beim Speichern dauerhaft übernommen.

Beim Schließen der App fragt der Bingo Generator nach, ob ungespeicherte Änderungen gespeichert werden sollen.

# English Description

## What is Bingo Generator?

**Bingo Generator** is a desktop application for creating custom printable bingo cards.  
It is designed for weddings, birthdays, parties, team events, icebreaker games and other occasions where guests should interact in a fun and informal way.

The app automatically creates bingo cards from custom questions and categories. Each card can be shuffled differently while still following the rules you define, such as category limits, required question types and layout settings.

## Typical workflow

### 1. Create categories

Questions are organized into categories, for example:

- Wedding couple
- School and university
- Travel
- Family
- Everyday life
- Hobbies

For each category, you can decide whether it is active and how many questions from that category should appear on each card. This makes it possible to ensure that certain topics are always included, while others are only used occasionally or as filler.

### 2. Edit questions

Each category can contain any number of questions.  
Questions should be written so that they complete the sentence:

> Find someone who ...

Example:

> is related to the wedding couple

On the bingo card, this becomes:

> Find someone who is related to the wedding couple.

### 3. Configure the game

The game section contains the main settings:

- Bingo grid size, for example 5 × 5
- Number of cards to generate
- Optional random seed for reproducible results
- Export format and output location

The optional random seed allows you to recreate the exact same set of cards later. If the field is left empty, the cards are shuffled differently each time.

### 4. Design the layout

The layout section lets you customize the appearance of the bingo cards:

- Title
- Subtitle
- Date
- Intro text directly above the grid
- Fonts, font sizes and colors
- Page margins
- Cell sizes and borders
- Corner images for a personal design

The included floral eucalyptus graphics can be used directly or replaced with your own images.

### 5. Check the preview

The preview shows a visual representation of the current layout on a white page.  
It helps you check spacing, headings, grid placement, images and text sizes before exporting.

The preview can be zoomed and scrolled. It is intended as a layout approximation; small differences may occur depending on the export format and printer.

### 6. Export

Finished bingo cards can be exported as:

- **Word document (.docx)** for further editing
- **PDF file (.pdf)** for direct printing or sharing

Before printing, the exported file should be checked once, especially when using custom images, very large font sizes or very small page margins.

## Languages

The user interface language can be changed through language files.  
Additional languages can be added by copying an existing language file, translating it and saving it under a new name.

## Saving

Settings and categories are stored locally.  
Category changes are kept while editing and are written permanently when saved.

When closing the app, Bingo Generator asks whether unsaved changes should be saved.
